# Função

Uma função é um bloco de código reutilizável que realiza uma tarefa específica. Em Python, as funções são definidas usando a palavra-chave `def`, seguida pelo nome da função e parênteses que podem conter parâmetros.

```python
def saudacao(nome):
    print(f"Olá, {nome}!")
```
Neste exemplo, a função `saudacao` recebe um parâmetro `nome` e imprime uma mensagem de saudação personalizada. Para chamar a função, você simplesmente usa seu nome seguido por parênteses, passando os argumentos necessários:

```python
saudacao("Maria")
```
Isso resultará na saída: `Olá, Maria!`
